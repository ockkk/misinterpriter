# 오역자들
영어 개발 문서에 익숙하지 않는 사람들이 모여 주 1회 영문서를 번역하는 스터디입니다. 
번역한 영문서를 공유하기도 하고 좀 더 보기 쉽게 하기 위한 웹페이지를 개발하였습니다. 
아직 부족한 부분이 많아 계속해서 업데이트 할 예정입니다.
